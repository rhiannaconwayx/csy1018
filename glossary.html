Term

Definition
Applications A piece of software available to code in.
Back End 'Behind the scenes' operations of coding etc. database connections
Body The body of your website is the centre section where all the information you want to display will go
Bootstrap Bootstrap is a framework that is open source and can be used by anyone
Classes Classes are used to reference elements in order to perform certain tasks within programming
CMS - Content Management System CMS is used to make modifications to a website or program
Comments Commenting is the action of adding snippets of information outside of your HTML or CSS code
CSS CSS stands for Cascade Style Sheet, this is the script that allows you to style your website
Contact Forms Contact forms can use a multitude of languages for validation checks etc. They are used to contact people via an online form
Database A database is a store of data that can be easily accessed and manipulated
Deprecated Code becomes deprecated when another piece of code has been created to improve efficiency
DIV The DIV tag is used to reference a section of HTML, it can also be used to group block-elements
External CSS An external CSS is used to style your HTML and design your website
Favicons Favicons are the small icons which show in the tabs or at the top of the page. They identify which page/website you're on
Fonts Fonts are the style of writing used within your website e.g. serif/sans-serif
Footer The footer is the display of information that is at the bottom of a webpage i.e. contact details/legalities etc.
Front End Front end is the opposite to back end. Front end design is the design that can be seen by the user e.g. the website itself
FTP FTP stands for File Transfer Protocol, this is used to transfer data between clients and servers over a network
Google Analytics Google Analytics is a web analytics software provided by Google that shows live data traffic e.g. the most popular page on your website
Google API Google API's allow for developers to use Googles services e.g. Google Maps/Translate within their website.
Google Fonts Google Fonts provide open source code that can be used to style the fonts in your website
Header A header is the information displayed at the top of a webpage
Hexadecimal Hexadecimal is a base 16 numbering system used to define colours when developing a website.
HTML HyperText Markup Language is the standard language used to create webpages
HTTP HyperText Transfer Protocol is used to define how data is formatted and transmitted e.g. it interacts with web servers
Hyperlink Hyperlinks are used to link external pages to your website e.g. when clicking the link you will view the webpage outside of your website
iFrame An iframe is an element within HTML that allows an external webpage to be embedded within your website.
Inline CSS Inline CSS refers to the CSS found in your HTML, usually it is at the top of the page between the style tags
Internal CSS Internal CSS refers to any styling made within the HTML script rather than the external CSS itself
JavaScript JavaScript is a computer programming language used widely in web development.
Meta Data Meta data is data contained within the header of your website, it cannot be viewed (aside from in the source code). It is extracted through XML
Nesting Nesting refers to putting one HTML element inside another.
Open Source Open Source refers to program code being made available to the general public.
PHP PHP (much like JavaScript) is a computer programming language used widely in web development.
Plug Ins A plug-in refers to a piece of third-party code that enriches the capabilities of a website e.g. Twitter plug-ins
Pseudo Element A Pseudo Element is used to add effects to specific pieces of code e.g. selectors
Repositories A repository is where you can save your website, it includes the pull and commit actions
Responsiveness Responsiveness refers to how smoothly your website runs, often when put under strain with a lot of traffic some websites can become slow.
Server A server is essentially used to provide data between computers. There are many different types of servers i.e. mail servers are used to provide email data
SFTP Secure File Transfer Protocol is a secure version of FTP which handles more sensitive data
Specification A specification is a document that lays out all the features and design of a piece of software/website as well as its capabilities
Styling The design/layout aspect of your website
Syntax Syntax refers to the rules that must be adhered too when programming
Tags Tags are used to encase an element to show where it starts and ends
Template A template is a file used to ensure the design of a website is consistent throughout.
URL A Uniform Resource Locator is the address of a website on the internet, it shows where a website can be found
Validation Validation means that a website returns no errors e.g. it has been validated and works properly
W3Schools Is a website to learn different languages such as PHP, JQuery, SQL, HTML. Other websites are available.
XML Extensible Markup Language allows users to create a custom markup language.
